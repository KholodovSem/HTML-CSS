Class and ID Selectors


ID
Мы можем давать нашим HTML - элементам уникальные идентификаторы, благодаря которым
мы сможем задавать стили для конкретного элемента.
Например:

HTML
<p id="author"></p>

CSS
#author {
  color: red
}

Тоесть, в HTML файле, мы присваеваем нужному тегу атрибут id, а в файле CSS,
если мы хотим обратиться к селектору по id мы используем #.

Class
Ещё один способ обращаться к конкретному элементу, это дать ему имя класса.
Нужному элементу присваеваем атрибут class и зада'м произвольное имя.
Это самый распространнёный и правильный подход.
Разница с id в том, что классы могут быть одинаковые сразу у нескольких элементов.
id всегда уникален для каждого элемента.
Пример:

HTML
<p class="number">1</p>
<p class="number">2</p>

CSS
.number {
  font-size: 28px
}
Теперь вместо решетки мы используем "."

У элемента может быть сразу несколько классов, разделяются они пробелом
<p class="number active">2</p>
