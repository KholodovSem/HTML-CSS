Selectors 

Селектор определяет, к какому элементу применять те или инные правила CSS.

Мы можем групировать правила CSS сразу для нескольких селекторов,
так например, если нам нужно задать font-family сразу для двух видов заголовков
h1, h2. Мы можем сделать это так:

h1, h2 {
  font-family: sans-serif;
}

Мы просто перечисляем селекторы через запятую и указываем общие для них правила.

Ещё одна возможность объеденения тегов в CSS.
Перечисление селекторов но уже без запятых.
Например, все абзацы на нашей веб-странице имеют размер шрифта - 22px.
Но параграф, который находиться в футере мы хотим сделать поменьше.
Для этого воспользуемся следующей возможностью:

footer p {
  font-size: 16px
}

Теперь браузер будет читать это как, параграф, который находится в элементе footer должен
иметь размер шрифта - 16px.
