CSS Grid

Грид представляет собой пересекающийся набор горизонтальных и вертикальных линий,
образующих колонки и строки.

fr - единица измерения, похожа на flex-grow. Проще объяснить на примере:
     У нас есть контейнер с шириной - 800px, мы создаём три колонки.
     У первой колонки ширина равна 200px, у второй тоже 200px.
     Если третей колонке мы зададим такуюже ширину, у нас останется назеполненое пространство.
     Тогда мы могли бы использовать fr в качестве единицы измерения, тогда третья колонка займёт всё
     оставшееся пространство. 

Keyword: 

min-content - ширина равна самому длиному слову.

max-content - содержимое вообще не будет переноситься на новую строку, ширина будет равна содержимому.

minmax() - принимает два параметра, минимальную ширину и максимальную. 
           Содержимое больше или равно менимальному значению, меньше или равно максимальному значению.

repeat() - принимает два параметра, первый - кол-во элементов для повтора, второй - их размер (ширина).

span - кол-во треков, которые должен занять элемент.

Grid property:

GridBox-property:
grid-template-columns - определяет кол-во колонок, указывать размер колонок можно в след. единицах:
                        px, проценты, fr (fraction).
                        Если в качестве параметров передать колонок больше чем элементов, то образуется
                        пустрое пространство равное указаной ширине, пока не появится соответсвующий
                        элемент.

column-gap - определяет размер отступа между колонками.

grid-template-rows - определяет кол-во рядов, указывать размер колонок можно в след. единицах:
                        px, проценты, fr (fraction).
                        Если в качестве параметров передать колонок больше чем элементов, то образуется
                        пустрое пространство равное указаной ширине, пока не появится соответсвующий
                        элемент.

row-gap - определяет размер отступа между рядами.

GridItem property:
grid-column-start - стартовая позиция элемента (линия колонки)

grid-column-end - конечная позиция элемента (линия колонки)

grid-column - сокращенный вариант записи двух предедущих свойств, принимает два параметра
              которые записываются через слеш: 1 / 4

grid-row-start - стратовая позиция элемента (линия ряда)

grid-row-end - конечная позиция элемента (линия ряда)

grid-row - сокращенный вариант записи двух предедущих свойств, принимает два параметра
           которые записываются через слеш: 1 / 4

grid-area - ещё более сокращенный варинат, который принимает в себя сразу четыре поряметра:
            grid-row-start, grid-column-start, grid-row-end, grid-column-end
            Пример: 1 / 1 / 4 / 4

1-------------2-------------3-------------4
|                                         |
|                                         |
|                                         |
|                                         |                                       
2                                         2                                                                                 
|                                         |
|                                         |
|                                         |
|                                         |
3                                         3
|                                         |
|                                         |
|                                         |
|                                         |
4-------------2-------------3-------------4

Нужно учитывать то, что колонок и рядов три, но отсчёт заканчивается на четвёрке.
Отсчёт всегда идёт с верхнего левого угла.
Если нам нужно начать отсчёт с противоположного угла, используем отрицательные числа.

Именование грид-линий
Именовать линии мы можем при создании либо столбцов, либо рядов, используя квадратные скобки.
Пример:

grid-template-column: [header-start] 1fr, 1fr, [header-end] 1fr

Выравнивание грид-элементов:

GridBox property: 
justify-items: выравнивание по горизонтали

align-items: выравнивание по вертикали

GridItem property:
justify-self: выравнивание по горизонтали. Stretch default (растягивает)

align-self: выравнивание по вертикали. Stretch default (растягивает)

Выравнивание сетки:
justify-content - выравнивает элементы по горизонтали

align-content - выравнивает элементы по вертикали


Управление порядком элементов:
grid-auto-flow: возможные варианты: 
                row (default), выкладывает элементы в ряд по порядку.
                column, тоже самое но для колонки.
                dance - заполняет возникшие пробелы между элементами, идёт в паре с предедущими свойствами.

order: работает аналогично свойству order-flexbox.
