HTML - HyperText Markup Language (Язык гипертекстовой разметки).
Используется для стурктурирования и описания содержимого страницы, с помощью элементов.
Таким образом у нас есть различные элементы, с помощью которых мы можем описывать 
различный по типу контент.

Строительные блоки из которых состоит каждый элемент HTML, на примере
(<p>HTML is a markup language</p>):

1. Открывающий тэг <p>
2. Содержимое элемента "HTML is a markup language"
3. Закрывающий тэг </p>

Всё вместе это называется элементом.

Существуют элементы, у которых нет контента и состоят они из одного тэга, являются 
самозакрывающимися тэгами, например:
<img/>
