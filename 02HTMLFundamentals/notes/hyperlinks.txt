Hyperlinks 

Ссылка <a></a>
Используется для перехода между страницами, либо для того чтобы создать привязку
к определённому контенту.
Существует две категории ссылок:
Первая - это навигационная категория, то есть они позволяют пользователю перемещаться по нашему сайту.
Вторая - это ссылки ведущие на внешние сайты, то есть за пределы нашего веб-сайта.
Создержит обязательный атрибут href, который говорит браузеру, куда нужно отправить
пользователя.
Необязательные атрибуты: 
target = открыть ссылку на новой вкладке или вместо текущей

Дополнительно:
Заглушка для ссылки - указать в href="#"
